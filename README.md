# LJT-Homepage

Personal academic homepage of **Junteng Liu**, PhD candidate at HKUST NLP Group.

## About

This site is built with the [Academic Pages](https://github.com/academicpages/academicpages.github.io) template, a Jekyll-based GitHub Pages theme for academic personal websites.

## Research

Research focuses on natural language processing and machine learning, with specific interests in:

- **LLM Reasoning and Reinforcement Learning**
- **Hallucination in Vision-Language Models (VLM)**
- **LLM Truthfulness and Interpretability**
